"Zenith's Trading Tool" (ZTT)
My largest and most involved program to date. Website side project programmed in JavaScript, HTML, and CSS, using the React framework for functionality, and Vite to package and deploy the site. Been under development since March 2025, and continues to be extremely fun to work on. Allows users to copy their inventory CSV string from the game "Celestial Caverns", load it in, and view and modify various statistics. Includes a trade tool, value chart, navigation bar, fully custom UI, background image/gif storing and optimization via a basic database, and much more. Even recently implemented data fetching from a wiki page via their API! Very proud of this project overall, it's even being actively used by a majority of active players. About Me

Hey! I'm Ben Rzasa, a PSU Computer Science undergrad, mediocre programmer, and enjoyer of video games. This website showcases some basic web development skills I've learned in the past year, as well as my work experience, projects I'm especially proud of, and a way to contact me. I always enjoy learning new topics related to CS, new languages, frameworks, operating systems, whatever gets my interest at a certain moment. May not have professional experience in any CS industry or be set up for a good internships like some of my peers, but I attempt to make up for it with genuine interest and willingness to try new projects.

"Arvik" packaging utility
C utility program meant to emulate basic features available on UNIX systems in the "ar" utility, and is also 100% compatible with "ar" extraction and compression, and other operations. Groups files together with a very specific header and gaps between them, into a ".arvik" file rather than ".ar". Also works with binary files. This one was fun to work on and test.
