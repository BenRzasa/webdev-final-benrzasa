Projects I'm Most Proud Of (so far...)

"Zenith's Trading Tool" (ZTT)
My largest and most involved project to date, with over 500 commits and 800 deployments to github-pages. Website side project programmed in JavaScript, HTML, and CSS, using the React framework for functionality, and Vite to package and deploy the site. Been under development since March 2025, and continues to be extremely fun to work on. Allows users to copy their inventory CSV string from the game "Celestial Caverns", load it in, and view and modify various statistics. Includes a trade tool, value chart, navigation bar, fully custom UI, background image/gif storing and optimization via a basic database, and much more. Even recently implemented data fetching from a wiki page via their API, after learning about it in this class! Very proud of this project overall, it's even being actively used by a majority of active players.

"Sieve of Eratosthenes"
Prime number generator programmed in Assembly. Takes user input, generates prime numbers up to an upper bound of 2 billion. Includes a script to test the runtime of the program with given text files.

Arcane Duel
C++ terminal-based two player card game utilizing templates for a doubly linked list deck data structure, populated by multiple types of cards. Includes basic health, buff, and damage functionality, as well as some cool ASCII art.

"Rockem" Client & Server
Basic client-server interface written in C, run from two separate areas. Can send files both ways, and has a few other functions such as checking if a file exists on the client or server, or simply reading the data directly.

Link: Click here to view Github repo

"Arvik" packaging utility
C utility program meant to emulate basic features available on UNIX systems in the "ar" utility, and is also 100% compatible with "ar" extraction and compression, and other operations. Groups files together with a very specific header and gaps between them, into a ".arvik" file rather than ".ar". Also works with binary files. This one was fun to work on and test.
